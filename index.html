Section 05

Tribal-area and rural systems, read together

The RHS publishes rural and tribal-area tables separately. Joining them yields a capacity index for each system (facilities in position over facilities required) and a disparity index, the tribal capacity relative to rural.

Capacity index C = in position ÷ required (1.0 = adequate)

Disparity index D = Ctribal ÷ Crural (below 1.0 = tribal areas worse off)

National values. 2019 is read from that report's HWC-integrated columns (same definition, renamed for one year). Endpoint values verified against the 2011 and 2021 source PDFs, table by table.
